# 🐍 Python Wiki Colaborativa

Bienvenidos a la Wiki Colaborativa de Python.

Este repositorio tiene como objetivo documentar el aprendizaje del curso de Python de forma colaborativa.

Aquí todos contribuimos como en un proyecto real.

---

# 🎯 Objetivo

- Aprender Python
- Aprender Git y GitHub
- Aprender documentación técnica
- Trabajar como en un proyecto profesional

---

# 📌 Reglas Principales

❌ No trabajar en main  
❌ No hacer push directo a main  
✅ Siempre trabajar con ramas  
✅ Siempre trabajar desde una Issue  
✅ Un Pull Request por tema  

---

# 🚀 Flujo de Trabajo

1. Tomar una Issue asignada
2. Hacer Fork del repositorio
3. Clonar tu fork
4. Crear una rama nueva
5. Hacer cambios
6. Push a tu fork
7. Crear Pull Request
8. Esperar revisión
9. Sincronizar fork
10. Repetir

---

# 📂 Estructura del Proyecto

El repositorio está organizado por módulos:
